# SMS Gateway Panel

A PHP-based SMS Gateway management panel using [Android SMS Gateway](https://github.com/capcom6/android-sms-gateway).

## Requirements

- PHP 8.2+
- Composer

## Installation

```bash
composer install
```

## Usage

Access the panel via your local server (e.g., Laragon) and navigate to `/login`.
